# svenhorvath-tools — Claude Code Plugin Marketplace

Zentraler Marketplace fuer Claude Code Plugins von Sven Horvath (Bauamt Vorderland).

## Plugins

| Plugin | Beschreibung |
|--------|-------------|
| [vaultkeeper](https://github.com/svenhorvath/vaultkeeper) | Obsidian Vault + Vorderland Brain Wissensmanagement |
| [protokoll-skill](https://github.com/svenhorvath/protokoll-skill) | Sitzungstranskripte zu Protokollen (Word) mit adaptivem Lernen |

## Installation

```bash
# Marketplace hinzufuegen (einmalig)
claude plugin marketplace add svenhorvath/claude-marketplace

# Plugins installieren
claude plugin install vaultkeeper
claude plugin install protokoll-skill
```
